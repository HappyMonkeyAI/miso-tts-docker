# Research

Reference material for **miso-tts-docker**. Not the source of truth — see
[CONTEXT.md](../CONTEXT.md) and [docs/adr/](../docs/adr/) for decisions.

## What belongs here

- External project comparisons
- Links worth revisiting
- Notes from exploring alternatives (LM Studio, INT4 quants, etc.)

## What does not belong here

- Final architecture decisions → `docs/adr/`
- User-facing setup → `README.md`
- Private chat exports or tokens

## Layout

```
research/
  README.md           # this file
  LINKS.md            # curated URLs
  templates/          # note templates
  github-projects/    # one file per upstream/reference repo
  notes/              # deeper comparisons (optional)
```
